# 知趣日报

知趣日报是一款集新闻、技术文章、福利于一体的阅读应用，内容分别由知乎、干货集中营和微信开放的api上获取的，采用了Material Design设计风格，MVP架构和Retrofit + RxJava网络请求框架等当前主流的开发技术，拥有漂亮的界面和稳定性能！欢迎大家安装和提问题，谢谢！
